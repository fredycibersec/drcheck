# Feature Guide

## Reset Button üîÑ

**Location:** Search form, next to "Check Reputation" button

**Functionality:**
- Clears the domain input field
- Hides all results and error messages
- Resets the application to initial state
- Automatically hides itself until next search

**When it appears:**
- Becomes visible after a successful domain analysis
- Hidden by default on page load

**Visual Design:**
- Gray gradient button (secondary style)
- Matches application color scheme
- Hover effect with slight color change

**Use Cases:**
- Start a fresh analysis without refreshing the page
- Clear sensitive domain information from view
- Quick reset during multiple domain checks

## PDF Export üìä

**Location:** Below overall reputation badge (after analysis completes)

**Functionality:**
- Generates professional PDF report
- Includes all analysis data from all sources
- Color-coded reputation indicators
- Formatted tables for easy reading
- Auto-downloads to your device

**Report Contents:**

1. **Header Section**
   - Title: "Domain Reputation Analysis Report"
   - Domain name analyzed
   - Overall reputation (color-coded)
   - Analysis date and time
   - Number of sources checked

2. **Per-Source Details**
   - Source name
   - Status (success/error/info)
   - Reputation score
   - Detailed metrics:
     * Malicious/Suspicious counts
     * IP addresses and locations
     * WHOIS information
     * Threat indicators
     * Port scans
     * And more...

3. **Footer**
   - "Generated by Domain Reputation Checker"
   - Timestamp

**File Format:**
- PDF (Portable Document Format)
- Letter size (8.5" x 11")
- Professional styling with tables
- Filename: `domain-reputation-{domain}-{YYYYMMDD}.pdf`
- Example: `domain-reputation-google.com-20260123.pdf`

**Visual Design:**
- Green gradient button (success color)
- Matches reputation "clean" theme
- Loading state: "‚è≥ Generating PDF..."
- Success: Auto-downloads file

**Technical Details:**
- Generated server-side using ReportLab
- Styled to match web application theme
- Colors: Cyan (#00d4ff), Blue (#0099cc)
- Tables with proper spacing and borders
- Handles complex nested data structures

## Troubleshooting

### Reset Button Not Showing
**Cause:** Analysis hasn't completed successfully
**Solution:** Wait for analysis to finish before reset appears

### PDF Export Fails
**Possible Causes:**
1. ReportLab not installed
   - Solution: `pip install reportlab`
2. Browser blocks download
   - Solution: Check browser download settings
3. Server error
   - Solution: Check Flask server logs

### PDF Contains "Unknown" Data
**Cause:** Some sources didn't return data
**Normal:** Not all sources work without API keys
**Solution:** Configure API keys for more complete data

## Design Philosophy

Both features follow the application's design principles:

1. **User-Friendly**
   - Clear visual feedback
   - Intuitive button placement
   - Self-explanatory icons and labels

2. **Professional**
   - Matches application theme
   - Consistent styling
   - High-quality PDF output

3. **Efficient**
   - Fast operations
   - Minimal clicks required
   - Smart state management

4. **Secure**
   - PDF generated server-side
   - No data persistence (privacy-first)
   - Clean state on reset
